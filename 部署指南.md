## 📝 部署步驟（5分鐘完成）

### 第 1 步：打開 Apps Script 編輯器

1. 點擊上面的 Google Sheets 鏈接
2. 在 Google Sheets 中，點擊頂部菜單：`擴充功能` → `Apps Script`
3. 會打開一個新的標籤頁（Apps Script 編輯器）

### 第 2 步：貼上代碼

1. 在 Apps Script 編輯器中，**刪除所有現有代碼**
2. 打開項目文件夾中的 `google-apps-script.js` 文件
3. **複製全部代碼**（從第一行到最後一行）
4. **貼上**到 Apps Script 編輯器中
5. 點擊 💾 **儲存**按鈕（或按 Ctrl+S）
6. 可以將項目命名為「HSK 學習系統 API」

### 第 3 步：部署為網頁應用程式

1. 點擊右上角的 `部署` 按鈕
2. 選擇 `新增部署作業`
3. 點擊「選取類型」旁的 ⚙️ 齒輪圖標
4. 選擇 `網頁應用程式`

**重要設定：**
- 說明：輸入「HSK API v1」
- **執行身分**：選擇 `我`
- **存取權限**：選擇 `所有人` ⚠️ **必須選這個！**

5. 點擊 `部署` 按鈕

### 第 4 步：授權

首次部署時需要授權：

1. 點擊 `授權存取權`
2. 選擇您的 Google 帳號
3. **可能會看到警告**：「Google 尚未驗證這個應用程式」
   - 點擊 `進階`
   - 點擊 `前往 HSK 學習系統 API (不安全)`
4. 點擊 `允許`

### 第 5 步：複製 URL

部署成功後：

1. 會顯示「網頁應用程式 URL」
2. **複製這個 URL**
3. URL 格式類似：
   ```
   https://script.google.com/macros/s/AKfycby.../exec
   ```
4. 您提供的 URL 是：
   ```
   https://script.google.com/macros/s/AKfycbybKYO7wJ3i9S12912i8I7sy5-kf-td9_s2tk-plblfL5tTHuxSm74fw6sjsQ6Qagdf/exec
   ```

### 第 6 步：在網站中測試

1. 打開 HSK 學習系統網站
2. URL 已經自動設置好了！
3. 點擊「⚙️ 設定」按鈕
4. 點擊「🔍 測試連接」
5. 如果顯示「✅ 連接成功」，就完成了！

完成部署後，測試以下功能：

1. **新增生詞**
   - 點擊「新增生詞」
   - 填寫表單

2. **查看生詞**
   - 點擊「瀏覽生詞」
   - 應該能看到剛才新增的生詞

   - 回到網站刷新頁面
   - 應該能看到更新

## 🔧 如果遇到問題

### 問題 1：連接失敗
**解決方法：**
- 確認 Apps Script 已經部署
- 確認存取權限設置為「所有人」
- 嘗試重新部署

### 問題 2：無法授權
**解決方法：**
- 點擊「進階」→「前往項目（不安全）」
- 這是正常的，因為這是您自己的腳本

### 問題 3：數據沒有同步
**解決方法：**
- 檢查 Google Sheets 的工作表名稱是否為「Sheet1」
- 點擊「⚙️ 設定」→「🔍 測試連接」查看錯誤訊息
- 刷新網頁（F5）

### 問題 4：URL 不正確
**解決方法：**
- 確保使用的是「網頁應用程式 URL」（部署 URL）
- 不是編輯器的 URL
- URL 應該以 `/exec` 結尾
